Introduction

Welcome to the Toxic API documentation. This API allows authorized bots and clients to securely fetch formatted user data based on Facebook IDs, Phone Numbers, or Email addresses. Access is strictly controlled via private API keys.

Endpoint: Lookup Search

GET https://toxiclabs.xyz/api

Query Parameters

ParameterTypeDescription
keyStringYour private authentication key (Required)
queryStringThe target's FB_ID, Phone, or Email (Required)

Integration Example (Python)

How your Telegram bot silently communicates with this API without exposing the key:

import requests

url = "https://toxiclabs.xyz/api"
params = {
    "key": "YOUR_SECRET_KEY",
    "query": "100018888067191"
}

response = requests.get(url, params=params)
if response.status_code == 200:
    print(response.json())
Security Notice: The Toxic Security Protocol is active. Any query targeting a `/protect` registered ID will automatically return a null response and log the requesting user's IP/Telegram ID.
