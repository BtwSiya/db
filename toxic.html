Toxic API Documentation

Enterprise-Grade Data Enrichment & OSINT Lookup Interface

Introduction

Welcome to the official documentation for the Toxic Security API. This RESTful API allows authorized systems, telegram bots, and custom clients to securely query our global database. You can retrieve highly formatted, accurate user data by submitting a target's Facebook ID, Phone Number, or Email address.

All API responses are formatted in standard JSON, making it easy to parse and integrate into any backend environment.

Authentication

Every request made to the Toxic API must include a valid API Key. Keys are issued exclusively to authorized administrators and routed through secured nodes.

Pass your API key using the key URL parameter in your GET request. Unauthorized requests will be immediately rejected with a 401 status code and logged in our system.

Lookup Endpoint

The primary endpoint for executing database queries against the OSINT records.

GET https://toxiclabs.xyz/api

Query Parameters

Parameter Type Required Description
key String Yes Your private authentication token.
query String Yes The target identifier (FB_ID, Phone Number, or Email).

Standard Error Codes

Our API uses standard HTTP response codes to indicate the success or failure of an API request.

Status Code Meaning Description
200 OK The query was successful and data was returned.
400 Bad Request Missing the required query parameter.
401 Unauthorized Invalid, missing, or revoked API Key.
404 Not Found No records matching the target query exist in the database.

Integration Examples

Python (Requests)

import requests

url = "https://toxiclabs.xyz/api"
params = {
    "key": "YOUR_SECRET_KEY",
    "query": "100018888067191"
}

response = requests.get(url, params=params)
if response.status_code == 200:
    print(response.json())
else:
    print(f"Error: {response.status_code}")

Node.js (Axios)

const axios = require('axios');

axios.get('https://toxiclabs.xyz/api', {
  params: {
    key: 'YOUR_SECRET_KEY',
    query: 'test@email.com'
  }
}).then(response => {
  console.log(response.data);
}).catch(error => {
  console.error(error.response.status);
});
